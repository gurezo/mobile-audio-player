# Mobile Audio Player Library (ionic-audio)

This library was generated with [Angular CLI](https://github.com/angular/angular-cli) version 17.3.0.

## Code scaffolding

Run `ng generate component component-name --project ionic-audio` to generate a new component. You can also use `ng generate directive|pipe|service|class|guard|interface|enum|module --project ionic-audio`.
> Note: Don't forget to add `--project ionic-audio` or else it will be added to the default project in your `angular.json` file. 

## Build

Run `ng build ionic-audio` to build the project. The build artifacts will be stored in the `dist/` directory.

## Publishing

After building your library with `ng build ionic-audio`, go to the dist folder `cd dist/ionic-audio` and run `npm publish`.

## Running unit tests

Unit tests for the workspace are run via the demo app: `ng test ionic-audio-demo` (Vitest).
